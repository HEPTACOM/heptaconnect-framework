## Additional development requirements

* Make
* Any debugging/coverage php extension like xdebug or pcov


## Contributing

Thank you for considering contributing to this package! Be sure to sign the [CLA](./CLA.md) after creating the pull request. [![CLA assistant](https://cla-assistant.io/readme/badge/HEPTACOM/heptaconnect-framework)](https://cla-assistant.io/HEPTACOM/heptaconnect-framework)


### Steps to contribute

1. Fork the repository
2. `git clone yourname/heptaconnect-framework`
3. Make your changes to master branch
4. Create your Pull-Request


### Check your changes

1. Check and fix code style `make cs-fix && make cs`
2. Check tests `make test`
3. Check whether test code coverage is same or higher `make coverage`
4. Check whether tests can find future obscurities `make infection`
